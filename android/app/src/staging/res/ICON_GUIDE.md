# Staging Flavor App Icons

Place your **staging** app icon files in the respective mipmap folders.

## Required Icon Sizes

Add an `ic_launcher.png` file in each folder with these dimensions:

- **mipmap-mdpi**: 48x48 px
- **mipmap-hdpi**: 72x72 px
- **mipmap-xhdpi**: 96x96 px
- **mipmap-xxhdpi**: 144x144 px
- **mipmap-xxxhdpi**: 192x192 px

## Design Tips for Staging Icons

- Add a "STAGING" or "STG" badge
- Use a different color scheme (e.g., orange/yellow tint)
- Consider a diagonal stripe or pattern overlay

## Quick Setup

You can use tools like:
- **Android Asset Studio**: https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html
- **App Icon Generator**: https://appicon.co/
- **flutter_launcher_icons** package (configure per flavor)
